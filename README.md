# trading

Class for technical indicators, data from FXCM python wrapper "fxcmpy".
    
    --- NaN values still included ---
    
    without signal:
    - SMA
    - EMA
    - MACD (uses EMA)
    - RSI (uses SMA)
    
    with signal: (inspiration for signal included)
    - Bollinger bands
    - Heiken-Ashi candles
    - Ichimoku Kinko clouds
